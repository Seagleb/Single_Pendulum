<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites


* npm

```sh
npm install npm@latest -g
```


### Installation

1. Clone the repo
```sh
git clone https://github.com/Seagleb/Single_Pendulum.git
```
2. Install NPM packages
```sh
npm install
```

<!-- USAGE EXAMPLES -->
## Usage

Start the web server
```sh
npm start
```
